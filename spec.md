# Specification

## Summary
**Goal:** Pre-populate the backend with sample menu items so the app displays real content out of the box.

**Planned changes:**
- Seed 8–12 sample menu items in the existing Motoko backend actor, covering at least 3 categories (e.g., Starters, Mains, Desserts, Drinks)
- Each seeded item includes a name, description, price, category, and availability set to true

**User-visible outcome:** The public Menu page and Admin Dashboard's Menu Management section display real menu items grouped by category immediately, without any manual admin action.
